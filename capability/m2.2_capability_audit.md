### 2.2 — T2: `mpmath.pslq` positive verification (known relations)

```
>>> from mpmath import mp, mpf, pslq, sqrt
>>> mp.dps = 50
>>> pslq([mpf(1), sqrt(2), 1 + sqrt(2)], tol=mpf(10)**(-40))
[1, 1, -1]                    # confirms 1·1 + 1·√2 + (-1)·(1+√2) = 0
>>> import mpmath as M
>>> pslq([M.log(2), M.pi, M.log(64)], tol=mpf(10)**(-40))
[-6, 0, 1]                    # confirms -6·log(2) + 0·π + 1·log(64) = 0 (since log 64 = 6 log 2)
```

**Status:** T2 PASS on known algebraic and log-relation identities. PSLQ correctly identifies the minimal-height relation.
